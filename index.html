A Lei de Proteção de Dados em 8 Pontos

O Regulamento Geral de Proteção de Dados (RGPD) estabelece várias diretrizes, incluindo:

1. Consentimento Claro para Transmissão de Dados a Terceiros: O RGPD torna obrigatório e claro o consentimento para a transmissão de dados pessoais a terceiros.

2. Proteção de Dados Sensíveis de Pessoas Falecidas: O regulamento também aborda a proteção de dados sensíveis pertencentes a pessoas falecidas.

3. Autorização para o Tratamento de Dados por Menores: O RGPD estabelece regras sobre a autorização do tratamento de dados por parte de menores, permitindo que os estados-membros definam uma idade mínima entre 13 e 16 anos.

4. Limitações nos Sistemas de Videovigilância: O RGPD impõe limitações aos sistemas de videovigilância, proibindo a vigilância em certos locais, como vias públicas, áreas privadas de clientes e áreas de trabalho.

5. Comissão Nacional de Proteção de Dados (CNPD): A CNPD é a autoridade nacional responsável pelo RGPD e pela sua implementação.

Autorização para a Transmissão de Dados Deve Ser Clara e Inequívoca

Antes do RGPD, em alguns casos, o consentimento para o uso de dados pessoais para fins comerciais era implícito. Com o RGPD, o consentimento deve ser claro e específico.

Menores podem Autorizar o Tratamento de Dados a partir dos 13 Anos

O RGPD permite que os menores autorizem o tratamento de seus dados pessoais a partir dos 13 anos, de acordo com a lei portuguesa.

Direito de Apagar Dados Pessoais

O RGPD permite que os consumidores solicitem a exclusão de dados pessoais a qualquer momento, desde que não sejam mais necessários para o propósito original ou tenham sido obtidos de forma ilegal.

Direitos em Caso de Morte

Em caso de morte, os direitos de acessar, corrigir ou apagar dados pessoais devem ser exercidos por alguém designado pela pessoa falecida ou pelos herdeiros.

Direito à Portabilidade dos Dados

O RGPD garante o direito à portabilidade dos dados, permitindo que os titulares dos dados recebam seus dados pessoais em um formato acessível.

Limitações nos Sistemas de Videovigilância

A legislação relacionada ao RGPD impõe restrições aos sistemas de videovigilância, especificando onde as câmeras podem ser instaladas.

Conservação Limitada de Dados Pessoais

Os dados pessoais devem ser mantidos apenas pelo período especificado na lei ou pelo tempo necessário para o propósito original. O direito de apagamento só pode ser exercido após o término desse prazo.

Tratamento de Dados para Finalidades Diferentes

O RGPD permite que entidades públicas processem dados pessoais para finalidades diferentes daquelas para as quais foram originalmente coletados.

Dicas para Reforçar a Proteção de Dados em 4 Passos

A implementação do RGPD afeta empresas e consumidores no dia a dia. Aqui estão algumas dicas:

1. Atualize suas Senhas Regularmente: Evite usar senhas previsíveis e altere-as regularmente, incluindo letras maiúsculas, minúsculas e números.

2. Leia os Termos e Condições: Leia atentamente os termos e condições, especialmente aqueles relacionados à coleta de dados pessoais, antes de aceitá-los.

3. Revise suas Opções de Compartilhamento: Você pode revisar e modificar suas opções de compartilhamento de dados pessoais a qualquer momento.

4. Remova Dados Pessoais em Dispositivos: Antes de descartar dispositivos que armazenam dados pessoais, certifique-se de excluí-los ou fazer uma cópia de segurança antes de redefini-los.
